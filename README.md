# 🏦 Bankaskipan GUI

## 🚀 Hvussu tú koyrir skipanina

1. Tryggja at Python er installerað á telduni.

2. Opna terminal og installera pakkar:

pip install flask oracledb python-dotenv

3. Klona projectið frá GitHub:

git clone https://github.com/tora142/SQL_bank_gui.git
cd SQL_bank_gui

4. Koyr forritið:

python app.py

5. Opna GUI:

Í terminalinum kemur eitt link, t.d.:
http://127.0.0.1:5000

Opna hetta í browser.
